## JF_item ##
simple usable custom items added some drug effects from https://github.com/it-scripts/it-drugs

## Installation ##
- add ensure JF_item to server.cfg should look something like this
ensure ox_lib

ensure qbx_core

ensure [qbx]

ensure JF_item

# Dependencys
- qbx_core or qb-core
- ox_inventory
- ox_lib
